# Laravel todo web app

## Setup
- `cp .env.example .env`
- Add database info to the [.env](./.env) file
### Linux & Mac
- `composer install && php artisan key:generate && php artisan migrate && php artisan serve`
### Win
- `composer install`
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan serve`
